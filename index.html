Grid Span

Grid Span is easy to use grid system based on CSS Grid Layout

Github

How the grid system works

Add class grid-container where you want to start using grid span. For example < main > tag

Add class grid-{number} on every parent element. You can define the number of columns you want to use for your website or application. The default is 12. You can configure your number in config.scss file.
