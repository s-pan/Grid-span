Grid Span

Grid Span is easy to use grid system based on CSS Grid Layout

Github

How the grid system works

Add class grid-container where you want to start using grid span. For example < main > tag 

                             
                  <main class="grid-container">
                    ......
                  </main>

Add class grid-{number of the columns} on every parent element. You can define the number of columns you want to use for your website or application in config.scss file. The default is 12. 

                                 
          <main class="grid-container">

            <div class="grid-12">
             ...children...
             </div>

             <div class="grid-12">
               ...children...
             </div>

           </main>

Now you are ready to define the cells(columns) which every child would comprise. 

         <main class="grid-container">

           <div class="grid-12">
             <div class="col-lg-1-4"></div>
             <div class="col-lg-5-8"></div>
             <div class="col-lg-9-12"></div>
           </div>

           <div class="grid-12">
             <div class="col-lg-4-10"></div>  
           </div>
           
         </main>
